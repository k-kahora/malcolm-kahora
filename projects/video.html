This demo showcases an application developed for efficient volumetrics of the company's NAS (Network Attached Storage). It features a graph displaying each cluster, aggregate, and volume. Users can interact with the bar graph to update a list and a pie chart below. The pie chart represents the total space in the containers, while the list displays the names of all selected nodes. Additionally, each node's color correlates with the cluster or aggregate it belongs to, facilitating easy identification.
